About Our Project

About Our Project

Our Proposal

Holidays are right around the corner. It’s time for people to put their travel plans on the calendars. Whether you’re trying to make that last minute flight for Thanksgiving dinner or planning months in advance for that beach vacation trip, what will prices look like? The goal of this project is to predict the flight price on a particular future date using a machine learning algorithm that will train historical data.

Finding Our Dataset

We started off with the idea of a project revolving around stock prices. Given that a lot of APIs are limited or they require a form of payment, we decided to go with flight prices using flight data from Kaggle. Additionally, we wanted to see if there was any correlation with fuel prices. To achieve this, we found data on the Bureau of Transportation Statistics site and EIA.

Dataset Limitations

With regards to the dataset, we were limited to one that contained only foreign airlines. We have concluded that this was due to limitations revolving around the availability of the data. Any domestic data that we identified required calling to a subscription API and we would run into potential legality issues through webscraping.

Our Process

Our Process

ETL Process

CSV Files from Kaggle were loaded and merged into Python via Jupyter Notebook.

Database

The data was normalized, cleaned, and then inserted into PostgreSQL from Jupyter Notebook.

Flask API

This tool was set up in order to connect the data to the site to calculate the flight prices.

Tableau

Using Tableau, visualizations were created to provide further insight into the Kaggle dataset.

Website Design

By utilizing tools like boostrap, Google fonts, and Coolors, this site was created based on a template from BootstrapMade.

JavaScript

Functionality of the button heavily relies on JavaScript in order to initiate the calculation on the backend and display the results on the frontend.

Machine Learning

Database was queried from PostgreSQL into a tailor-made pandas dataframe for the KNN Regression Model in order to create a price predictor.
